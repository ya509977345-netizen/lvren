配置说明

如果邮件发送失败，请检查以下配置：

1. 服务器邮件服务
# CentOS/RHEL sudo yum install sendmail sudo systemctl start sendmail sudo systemctl enable sendmail # Ubuntu/Debian sudo apt-get install sendmail sudo systemctl start sendmail sudo systemctl enable sendmail
2. PHP配置 (php.ini)
[mail function] SMTP = smtp.example.com smtp_port = 587 sendmail_from = noreply@example.com sendmail_path = /usr/sbin/sendmail -t -i
3. 替代方案 - 使用第三方邮件服务

您也可以考虑使用以下服务来发送邮件：

  • SendGrid - 免费100封/天
  • Mailgun - 免费1000封/月
  • AWS SES - 低成本，高可靠性
  • 阿里云邮件推送 - 国内用户友好
